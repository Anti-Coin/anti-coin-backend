## 2. Rules
1. Archive 파일은 append-only 원칙을 따른다.
2. 기존 ID(`D-xxx`, `A-xxx`, `B-xxx`, `C-xxx`, `TD-xxx`)는 변경하지 않는다.
3. 활성 문서에서 제거된 상세 이력은 반드시 Archive 링크를 남긴다.

## 3. When To Read Archive
1. 과거 결정의 원문 맥락이 필요할 때
2. 특정 작업의 상세 변경 파일 히스토리가 필요할 때
3. 회고/감사(audit) 목적의 근거 추적이 필요할 때
